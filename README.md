# Homey Flow Architect (HFA)

Building a "Co-Pilot" for the Advanced Flow editor on the Homey Pro 2026.

## Core Philosophy
- **Creation over Execution:** We are building a "Co-Pilot" for the Advanced Flow editor, not a voice assistant.
- **In-Doors Traffic:** All logic is deployed to the Homey Pro 2026 for local execution.
- **Schema-First:** Every action must be validated against the Homey Web API schema to prevent UI corruption.

## Setup
1. Clone the repository
2. Run `npm install`
3. Start coding!

See `system_prompt.md` for the full Agent Standard Operating Procedure.
